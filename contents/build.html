Build¶

  1. Create a copy of the project from its Github repository

  2. Install CocoaPods if you haven’t done it already

  3. Download Pod libraries at the project directory by executing pod install

    $ pod install

  4. Open GiottoDataViewer.xcworkspace (not GiottoDataViewer.xcodeproj) and build
