Quick Test with Demo Scripts¶

The page explains how you can test the GiottoDataViewer with minimal steps.

  1. Install BuildingDepot
  2. Copy pythons scripts in demo_script folder to a server hosting BuildingDepot.
  3. Run add_demo_sensors.py to register sensors in BuildingDepot.
  4. Run dummy_sensor.py. This script sends dummy data to BuildingDepot.
  5. Build GiottoDataViewer as described in Build.
  6. Configure BuidingDepot parameters in the iOS app as described in Setup.
  7. Put Room1 in the location emulation in the Setting tab of GiottoDataViwer
  8. You should see a list of devices in the Devices tab. If it does not work, check the Log tab to see what is the problem.
