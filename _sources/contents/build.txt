==================
Build
==================

#. Create a copy of the project from its Github repository
#. Install CocoaPods_ if you haven't done it already
#. Download Pod libraries at the project directory by executing pod install

   .. code-block:: bash

   	$ pod install

#. Open **GiottoDataViewer.xcworkspace** (not GiottoDataViewer.xcodeproj) and build

.. _CocoaPods: https://cocoapods.org
